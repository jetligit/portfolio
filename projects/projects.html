Terrain Aware Pathfinder

Developed an A*-based pathfinding system that computes the optimal route across real-world terrain using satellite map colors, elevation data, and terrain-dependent movement costs. The program parses a 395x500 terrain image, elevation grid, and control-point file to determine the fastest traversal path and overlays the result on the original map. Implements efficient graph search with heuristics, custom cost functions, and pixel-level geospatial calculations.
